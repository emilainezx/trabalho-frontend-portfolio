## Sobre

Este projeto consiste em um **portfólio web responsivo**, desenvolvido por estudantes do curso de **Análise e Desenvolvimento de Sistemas (ADS)**.  

O objetivo é apresentar projetos acadêmicos e estudos práticos, aplicando conceitos de **HTML, CSS, Bootstrap e JavaScript**, com foco em organização, responsividade e boas práticas de desenvolvimento web.

---

## Primeiros Passos

O projeto pode ser executado localmente de forma simples, sem necessidade de instalação de servidores ou bibliotecas adicionais.

Basta acessar o arquivo principal pelo navegador para visualizar o portfólio.

---

## Pré-requisitos

Para utilizar o projeto, é necessário apenas:

- Um navegador web atualizado (Google Chrome, Microsoft Edge, Firefox, entre outros)
- Um editor de código para visualização ou edição dos arquivos (opcional)
- Manter a estrutura de pastas do projeto organizada

---

## Instalação

1. Faça o download ou clone este repositório  
2. Verifique se a estrutura de arquivos está correta:
   - `index.html`
   - `style.css`
   - pasta `img/` com as imagens dos projetos
3. Abra o arquivo `index.html` em um navegador web

Após isso, o portfólio estará pronto para uso.

---

## Como Usar

1. Abra o projeto no navegador  
2. Utilize o menu de navegação para acessar as seções  
3. Visualize os projetos apresentados nos cards  
4. Clique nos links disponíveis para acessar repositórios ou conteúdos relacionados  
5. Utilize os ícones no rodapé para acessar o GitHub dos desenvolvedores  

---

## Tecnologias Utilizadas

- **HTML5** – Estruturação do conteúdo  
- **CSS3** – Estilização e layout visual  
- **Bootstrap 5** – Responsividade e componentes prontos  
- **JavaScript** – Interatividade e comportamento dinâmico  
- **Font Awesome** – Ícones utilizados na interface
